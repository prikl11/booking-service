## Database Schema

### users

| Поле       | Тип          | Ограничения       |
| ---------- | ------------ | ----------------- |
| id         | integer      | PK, autoincrement |
| first_name | varchar(50)  | NOT NULL          |
| last_name  | varchar(100) | NOT NULL          |
| email      | varchar(150) | NOT NULL          |
| phone      | varchar(40)  |                   |
| password   | varchar      | NOT NULL          |
| image_url  | varchar(255) | NULL              |
| is_admin   | boolean      | DEFAULT False     |
| created_at | date         |                   |
| updated_at | date         |                   |

### hotels

| Поле        | Тип          | Ограничения       |
| ----------- | ------------ | ----------------- |
| id          | integer      | PK, autoincrement |
| name        | varchar(150) | NOT NULL          |
| description | text         |                   |
| created_at  | date         |                   |
| updated_at  | date         |                   |

### hotel_staff

| Поле     | Тип     | Ограничения             |
| -------- | ------- | ----------------------- |
| user_id  | integer | PK, FK → users.id       |
| hotel_id | integer | PK, FK → hotels.id      |
| role     | text    | NOT NULL (ENUM на беке) |

### rooms

| Поле        | Тип          | Ограничения                |
| ----------- | ------------ | -------------------------- |
| id          | integer      | PK, autoincrement          |
| hotel_id    | integer      | FK → hotels.id             |
| name        | varchar(150) | NOT NULL                   |
| description | text         |                            |
| price       | numeric      | DEFAULT 0                  |
| discount    | numeric      | NULL (коэффициент 0.0–1.0) |
| is_active   | boolean      | DEFAULT True               |
| created_at  | date         |                            |
| updated_at  | date         |                            |

### services

| Поле       | Тип          | Ограничения       |
| ---------- | ------------ | ----------------- |
| id         | integer      | PK, autoincrement |
| name       | varchar(100) | NOT NULL          |
| image_url  | varchar(255) |                   |
| created_at | date         |                   |
| updated_at | date         |                   |

### hotel_services

| Поле       | Тип     | Ограничения          |
| ---------- | ------- | -------------------- |
| hotel_id   | integer | PK, FK → hotels.id   |
| service_id | integer | PK, FK → services.id |

### room_services

| Поле       | Тип     | Ограничения          |
| ---------- | ------- | -------------------- |
| room_id    | integer | PK, FK → rooms.id    |
| service_id | integer | PK, FK → services.id |

### hotel_images

| Поле      | Тип          | Ограничения       |
| --------- | ------------ | ----------------- |
| id        | integer      | PK, autoincrement |
| hotel_id  | integer      | FK → hotels.id    |
| image_url | varchar(255) | NOT NULL          |

### room_images

| Поле      | Тип          | Ограничения       |
| --------- | ------------ | ----------------- |
| id        | integer      | PK, autoincrement |
| room_id   | integer      | FK → rooms.id     |
| image_url | varchar(255) | NOT NULL          |

### cart

| Поле       | Тип     | Ограничения       |
| ---------- | ------- | ----------------- |
| id         | integer | PK, autoincrement |
| user_id    | integer | FK → users.id     |
| created_at | date    |                   |
| updated_at | date    |                   |

### bookings

| Поле            | Тип       | Ограничения       |
| --------------- | --------- | ----------------- |
| id              | integer   | PK, autoincrement |
| cart_id         | integer   | FK → cart.id      |
| room_id         | integer   | FK → rooms.id     |
| total_price     | numeric   | NOT NULL          |
| people_quantity | integer   |                   |
| status          | text      | ENUM на беке      |
| comments        | text      | NULL              |
| arrival_date    | timestamp | NOT NULL          |
| departure_date  | timestamp | NOT NULL          |
| created_at      | date      |                   |
| updated_at      | date      |                   |
